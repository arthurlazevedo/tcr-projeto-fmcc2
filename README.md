## Rodar facilmente

Navegue até o diretório onde o repositório está localizado e rode:

```shell
$ python3 -m http.server
```

ou, caso esteja no windows:

```shell
$ python -m http.server
```

Depois, digite `localhost:8000` no seu navegador de preferência
